# Quality Assistance - Nightwatch (Parallel Testing Firefox, Chrome, Safari)

Nightwatch.js is a node.js framework which utilizes selenium webdriver for continuous integration and automation. The following setup will run your test suite in parallel via a shell script.

### Prerequisites

Node.js - https://nodejs.org/en/
npm - https://www.npmjs.com/get-npm

`cd nightwatch`
`npm install nightwatch`

### Getting Started

`sh run.sh`

 The above script will remove any previous files saved in the reports and screenshots folders before executing the nightwatch test command.

### Documentation

Nightwatch.js - http://nightwatchjs.org/
Github repo - https://github.com/nightwatchjs/nightwatch
